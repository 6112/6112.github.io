to-do list

A simple web application for keeping track of things to do. Allows simple editing, adding, removing, and moving around of a list's element. Uses localStorage to save the items even when you close your browser, and HTML5 drag-and-drop to allow you to move items in the list.

Ciel

Ciel is a simple module for syntax highlighting in any language. It is written in JavaScript and executes on the client-side after page load.

Languages can easily be added by specifying, in the source code, the keywords, and other syntax of the desired language.
