To-do List

A simple web application for keeping track of things to do. It allows simple editing, adding, removing, and moving around of a list's element. It uses localStorage to save the items even when you close your browser, and HTML5 drag-and-drop to allow you to move items in the list.

Ciel

A module for syntax highlighting in any language. It is written in JavaScript and executes on the client-side after page load.

Languages can easily be added by specifying, in the source code, the keywords, and other syntax of the desired language.

Fluid Simulator

A Java application for real-time, 2D fluid simulation. It uses a particle system to mimic the behaviour of a liquid inside a bottle. You can use your mouse to pick up some of the fluid, and you can change some of its physical properties.

js-curses

ncurses is a C library that allows developers to create command-line applications that use text as their user interface, and allows advanced formatting and keyboard interaction inside a terminal.

js-curses is a similar library that aims to allow to create applications that feel like native command-line ncurses programs, in the web browser.
